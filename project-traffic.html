About the Project

This project uses the YOLO (You Only Look Once) deep learning model to detect and classify various traffic signs from images and videos. The system can identify traffic symbols such as speed limits, stop signs, and warning signs in real-time, making it valuable for autonomous driving systems and road safety applications.

The project demonstrates expertise in computer vision, deep learning, and real-time object detection, with practical applications in intelligent transportation systems.
